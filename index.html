HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<th> Table Heading The title to rows or columns.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Value Description Related Tag(s)
scope Header Scope col / row / colgroup / rowgroup Identifies whether a cell is a header for a column, row, or group of columns or rows. <th>
colspan Column Span [int ≥ 1] Defines how many columns a td or th element should span. (For improved accessibility, when creating table headings that span multiple columns, include the scope="colgroup" attribute in <th>.) <td>, <th>
rowspan Row Span [int ≥ 1] Defines how many rows a td or th element should span. (For improved accessibility, when creating table headings that span multiple rows, include the scope="rowgroup" attribute in <th>.) <td>, <th>
